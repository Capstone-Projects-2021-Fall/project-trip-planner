Collaborative Trip Planner

About

A web application that allows multiple users to collaborate on a trip itinerary in real time. Changes made by any one user will update the trip itinerary for every user who is currently viewing the trip. Any text entered will be parsed by the application for keywords such as locations, events, and times. These keywords will then be used to search the web for possible additions to the trip itinerary similar to existing events already added by the user. Users can approve these additions and they will be added to the itinerary. Trip itineraries can be saved to be edited later or shared with other users on the website as inspiration for their own trip.

System Overview

Trip Planner is a web application with social networking features that enables users to collaborate on a trip itinerary in real time. Users will be able to quickly sign up using their G-mail account which allows them to connect with contacts from their email. The platform will allow users to express themselves by liking shared trip itineraries and following other users. Users will connect by contributing to a trip by suggesting locations and events. The application will also provide automated suggestions based on similar trips and events happening around that time. Itineraries will have features that allow users to follow the post to contribute at a later time or use as an inspiration for a personal trip. Users will also be able to add photos and videos of visited locations so that future travelers can have a reference.
